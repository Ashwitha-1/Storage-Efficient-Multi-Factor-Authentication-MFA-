# 🔐 Storage-Efficient Multi-Factor Authentication (MFA)

A full-stack authentication system designed to enhance user security through Multi-Factor Authentication (MFA) while optimizing storage efficiency and authentication performance.

---

## 📋 Overview

Storage-Efficient MFA is a secure authentication platform that combines modern web technologies with multi-factor verification mechanisms to provide robust account protection. The system focuses on reducing storage overhead while maintaining high security standards and a seamless user experience.

---

## 🚀 Features

* 🔐 Secure User Registration & Login
* 🔑 OTP-Based Multi-Factor Authentication
* 🛡️ JWT Authentication & Authorization
* 🔒 Password Hashing & Secure Storage
* 📧 Email Verification Support
* 💾 Storage-Efficient Authentication Records
* ⚡ Fast Authentication Workflow
* 📱 Responsive User Interface
* 🚪 Secure Session Management

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/charan2228/storage-efficient-MFA.git
```

### Navigate to the Project

```bash
cd storage-efficient-MFA
```

### Install Dependencies

```bash
npm install
```

### Start the Application

```bash
npm start
```

### Run Backend Server

```bash
node server.js
```
